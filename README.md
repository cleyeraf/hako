# hako (個人用Webツール集)

`hako` は、日常の開発業務や個人利用でよく使う小ツールをブラウザ内で完結させる、安全・軽量な静的Webツール集です。

## 🔒 セキュリティ・プライバシー方針 (絶対厳守)
- **すべての処理はブラウザ内で完結します。** 入力されたデータが外部サーバーへ送信されることは一切ありません。
- **外部へのネットワーク通信は一切行いません。** 外部CDN（CSS/JS）、外部フォント、外部APIなどの呼び出しを排除しており、オフラインでも完全に動作します。
- ソースコードおよびコミット履歴に、メールアドレスや本名などの個人情報・機密情報を混入させないよう注意してください。

## 🛠️ 搭載ツール一覧
1. **[BASE64エンコード・デコード](tools/base64/index.html)**
   - テキストをBASE64形式にエンコード・デコードします。
   - 日本語（マルチバイト文字）や URL-safe 形式（RFC 4648）に対応しています。
2. **[URLエンコード・デコード](tools/url-encode/index.html)**
   - テキストをURLエンコード（パーセントエンコーディング）形式に変換、またはデコードします。
   - encodeURIComponent/encodeURIの切り替えや、スペースの変換（%20または+）に対応しています。
3. **[ランダム文字列生成](tools/random-string/index.html)**
   - 暗号論的に安全な乱数生成器 (`window.crypto.getRandomValues`) を使用したランダム文字列・パスワード生成器です。
   - 偏りのないリジェクションサンプリングを採用しています。
4. **[日付計算](tools/date-calculator/index.html)**
   - 指定した日時からの日付加減算（X日後・前など）や、2つの日付の間の差分（日数・時間など）を計算します。
5. **[QRコード生成](tools/qr-code/index.html)**
   - 入力したテキストやURLからQRコードをリアルタイムで生成します。
   - 誤り訂正レベル（L / M / Q / H）を選択可能。SVG形式でのダウンロード・クリップボードへのコピーに対応しています。
   - QRコード生成ライブラリ（`qrcodegen.js`）はローカルにバンドルされており、外部通信は一切行いません。
6. **[ROCK — CAVE FLIGHT](games/rock/index.html)**
   - 障害物を避けて洞窟を飛び続ける、スリリングな障害物回避ゲームです。
   - ランキングデータはブラウザの `localStorage` に安全に保存されます。

## 🚀 ローカルでの実行・テスト方法
本プロジェクトは純粋な静的ファイル（HTML / CSS / JS）のみで構成されています。
リポジトリをクローンした後、ルートの `index.html` をブラウザで直接開くか、ローカルサーバーを起動してアクセスしてください。

### ローカルサーバー起動例 (Python)
```bash
python3 -m http.server 8000
```
起動後、ブラウザで `http://localhost:8000` にアクセスしてください。

## 📦 GitHub Pages への公開方法
1. GitHub でリポジトリを `public` で作成します（個人情報を含まないことを再確認してください）。
2. リポジトリ設定 (`Settings > Pages`) に移動します。
3. `Build and deployment > Source` で **Deploy from a branch** を選択します。
4. `Branch` に `main` および `/ (root)` を指定して **Save** をクリックします。
5. 数分後、`https://{username}.github.io/hako/` で公開されます。
